Process Scheduling Simulator

The Java Swing-based CPU Scheduler desktop application I've developed serves as a valuable tool for understanding and simulating various scheduling algorithms used in operating systems. This application enables users to explore and analyze First-Come-First-Serve (FCFS), Shortest Job First (SJF), and Round Robin (RR) algorithms. Through an intuitive interface, users can input different processes, set burst times, and observe the outcomes of each scheduling approach.
